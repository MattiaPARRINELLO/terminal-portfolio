## 🎯 Présentation
Terminal Portfolio réinvente le portfolio traditionnel en une interface CLI interactive inspirée du terminal VS Code. L'utilisateur explore les sections en tapant des commandes — `projects`, `help`, `clear`, `a propos` — comme s'il naviguait dans un vrai terminal. Un easter egg `sudo rm -rf` ajoute une touche d'humour geek.

---

## ✨ Fonctionnalités
- ✅ Interface CLI interactive avec historique des commandes
- ✅ Commandes : `projects`, `help`, `clear`, `a propos`, et plus
- ✅ Easter egg `sudo rm -rf`
- ✅ Police Fira Code et icônes Font Awesome 5
- ✅ Déploiement sur [sobot.fr](https://sobot.fr)
- ✅ Design retro/vintage fidèle au thème VS Code

---

## 🚀 Installation
```bash
git clone https://github.com/MattiaPARRINELLO/terminal-portfolio.git
cd terminal-portfolio
# Ouvrir index.html dans un navigateur
```

---

## 📖 Utilisation
Ouvrez `index.html` dans votre navigateur. Tapez `help` pour voir la liste des commandes disponibles. Explorez les projets avec `projects`, découvrez l'auteur avec `a propos`, ou amusez-vous avec `sudo rm -rf`.
